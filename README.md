# Elõkészítés
- Tíz elemû egész számot tartalmazó tömb létrehozása
- A tömb elemei: 12, 7, 2, 4, 45, 99, 1, 8, 95, 50
- A tömb az összes metódusban látható legyen
- Tömb kiírás metódus létrehozása
- - Nem ad vissza értéket és nincs paramétere
  - Tömb elemei ,-vel elválasztva egy sorban a consol-ra írja
## Összegzés tétele:
- Összegzés metódus létrehozása
- Összegzés tétele a tömbön
- Eredmény kiírása a consol-ra
- Hány 3-mal osztható elem van a tömbben?
- Eredmény kiírása a consol-ra
## Eldöntés:
- Metódus létrehozása
- Van-e a tömbben 69?
- kiírni hogy van vagy nincs
## Kiválasztás
- Metódus létrehozása
- Hányadik az 50-es szám?
- Írjuk ki az indexét
## Keresés (lineáris)
- Keresés metódus létreh.
- Van e benne 50 akkor hányadik.
## Maximum 
- Metódus létreh.
- Kiírni a legnagyobb elemet.
